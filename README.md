# multi_car_simultion

## Demo

<img src="./multi_car_map.gif" width="600">

### Contents

- 자동차 시뮬레이션 및 자동차 운전 점수 측정
- 출발지 부터 도착지 까지 주행 시뮬레이션
- 다른 자동차들과의 충돌 , 신호 위반 등을 측정하여 점수 계산
- ``A``, ``D`` , ``W``, ``S`` 키를 통해 회전, 직진 및 후진 조절
- 사용 기술 : ``OpenGL``, ``C++``
